Positioning

Below the ruler, in the exercise section, is a div with the id "testDivPosition." Add a rule selecting this div by id to the style section. Add the following properties to this rule and note what happens.
  1. Set the "position" property to "relative." This tells the browser to move the element relative to its existing position in the normal flow.
  2. Set the "left" property to a value of 100px. What happens?
  3. Remove the "left" property and set a "right" property to a value of 100px. What happens?
  4. Remove the "right" property and set a "top" property to a value of 100px.
  5. Keep the "top" property in place and add a "left" property with a value of 200px.
  6. Remove the "top" and "left" properties.
  7. Set the "position" property to "absolute." This tells the browser to move the element to an absolute distance away from the edge of the browser window.
  8. Set the "top" property to 0px.
  9. Retain the "top" property and set a "right" property with a value of 0px.
  10. Change the value of the "right" property to 100px.
  11. Remove the "top" property
  12. Add a "bottom" property with a value of 0px.
  13. If you finish all these steps, try some other property value pairs, and see how you can position the element.
